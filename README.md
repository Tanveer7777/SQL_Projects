-- Retail Sales Data Analysis Using SQL

This project explores and analyzes retail sales data using SQL. It aims to uncover key insights about customer behavior, sales trends, and product performance, which can help in making data-driven business decisions.

-- Project Overview

Using a table named retail_sales, this project answers several business-critical questions, such as:

Total sales and number of unique customers

Product category analysis

High-value transactions

Customer segmentation by gender and age

Date-specific and category-specific trends

The project demonstrates practical SQL skills through the use of:

SELECT, WHERE, GROUP BY, ORDER BY, COUNT, SUM, AVG, DISTINCT

Conditional filtering

Date range queries

Aggregation and grouping

-- SQL Query List

Total Sales Count

Unique Customer Count

Available Product Categories

Daily Transactions Query

Category + Quantity Filtered Sales

Category-Wise Sales Aggregation

Average Age of Customers (Beauty Category)

High-Value Transactions (>1000)

Gender-wise Transaction Count by Category

Top 5 Customers by Sales

Unique Customers per Category
